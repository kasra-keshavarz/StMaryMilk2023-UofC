README

As per page 13 of the natural flows report, gauge 06167500 uses measured data from gauge 06166000.

As per page 21 of the natural flows report, gauge 06171000 used measured data from gauge 06169500.

USGS data for gauges 06166000 and 06169500 were downloaded and provided in this folder. The R script "write_observations.R" 
downloads the rest of the USGS data associated with each DNRC naturalized gauge.
